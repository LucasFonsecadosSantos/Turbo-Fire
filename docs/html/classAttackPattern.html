Detailed Description

AttackPattern is the protocol for attack order communication between master and zombies host.

Author
Lucas Fonseca dos Santos (lucas.nosp@m.@lcf.nosp@m.compa.nosp@m.ny.c.nosp@m.om.br)
Date
2019-04-26
Version
1.0.002 The AttackPattern object is used to organizes the attack operation order communications between master and zombies host. It contains the whole attack informations as:
  • Target host ip source;
  • Target port;
  • Connection time out;
  • Attack time out;
  • Zombie thread amount;
  • Protocol will be used;
  • The content of packages (message); These informations are very necessary to zombie host makes the attack to some server.
